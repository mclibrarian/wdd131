American Girl History

Site Name: American Girl History

Reason for selecting: This name represents a website that contains information and history about American Girl dolls.

Optional domain availability: americangirlhistory.com

Site Purpose

The site provides a way for people who are interested in American Girl dolls to find more information about them by providing details about the dolls and their stories and a form to request more specific information that may not be available.

Scenarios

1: What is the most recently released American Girl doll?

2: How do I get an American Girl doll?

Color Schema

Color 1 (main color): rgb(173, 21, 67), a color very similar to the main American Girl color, for use on the header background, footer background, and heading text colors.

Color 2 (accent color): antiquewhite, for use as a display text color and a heading background color.

Typography

Font 1: Noto Serif Display, for display type

Font 2: Quicksand, for main type
